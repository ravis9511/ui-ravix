# Avatar

Avatar is used to represent a user or group of users using an image. It can also convey additional information about the user like login status, busy, away etc. The avatar can also be used as a link to a profile page.

**Accessibility Note:**  Pay close attention to the proper utilization of aria-label, aria-hidden, and alt attributes when working with different avatar elements. For links, you can enhance the accessibility experience by incorporating supplementary information within the aria-label attribute, effectively conveying the context of the associated image, icon, or initials and set aria-hidden="true" on the associated elements inside the link.
