| Stylized Button            | Yes  | Yes              | Yes  | Yes | Yes              |

ravixUI uses the semantic `<button>` element for all non link buttons and `<a>` element for link buttons. The size of the buttons meet the minimum target size for interactive elements as [defined by the WCAG 2.1 specification.](https://www.w3.org/WAI/WCAG21/Understanding/target-size.html). The minimum target size for all the buttons in this context, i.e., with the fonts used, the outer containers of the buttons etc. is 44 x 44 px. 

While the 44px X 44px target size is important, another important factor to note is that there should be ample space between clickable areas of buttons placed next to each other.
