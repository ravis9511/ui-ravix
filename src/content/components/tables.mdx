# Table

Tables are used to display data in a structured and organized manner. It is the most common way to display a lot of structured data in a web page. The tables in ravixUI comes with striped rows out of the box to make it easier to read.


## Table Specification

A table has three main parts :

- The table container : The container  div or section that wraps the table
- The table header : The header row that contains the column names
- The table body : The body of the table that contains the data
